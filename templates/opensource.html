La realizzazione di #italiasicura

#italiasicura nasce grazie all'unione di competenze multidisciplinari, ispirata in particolare a:

  • integrazione e riuso di dati aperti
  • utilizzo di software open source e tecnologie aperte
  • produzione e distribuzione come software open source

i principali criteri considerati nella costruzione dell'applicazione #italiasicura sono:

  • consentire una navigazione semplificata della mappa evitando la logica GIS di "abilita/disabilita layers" e utilizzando invece il contesto locale per la generazione delle informazioni
  • spingere la risoluzione spaziale fino alla scala sub-comunale (località) e quindi il più possibile prossime ai cittadini
  • accedere alle informazioni esplorando l'area di interesse con il mouse e ottenere gli ulteriori dettagli con un solo click
  • privilegiare elevate capacità di socializzazione delle informazioni

#italiasicura e l'Open Source

L'applicazione è stata costruita impiegando esclusivamente software, librerie e tool open source:

  • Backend & tools
    • GNU/Linux Ubuntu Server
    • PostgreSQL Open Source Database / PostGIS GeoSpatial database
    • Boundless GeoServer Server Geografico che serve all'applicazione Mappe, Tiles e Features Vettoriali
    • WEKA 3 libreria Java e tool per l'analisi dati
    • QGIS tool per la gestione ed il processamento di file e formati dati geospaziali
    • Apache Jena libreria Java per gestire serializzazione in formato RDF e processore SPARQL
    • OpenRefine tool per l'esplorazione ed il processamento di dati tabulari complessi
    • Apache Tomcat application server container per java web-application
    • Jersey Libreria Java per gestire Web Services RESTful
    • Apache HTTPD Server server HTTP
    • PHP PHP scripting language
    • Twitter-API-PHP PHP wrapper della API Twitter
  • Frontend & Software Libraries
    • AngularJS framework javascript per la creazione di web application dinamiche
    • OpenLayers 3 libreria per la gestione della mappa
    • Twitter Bootstrap framework e componenti HTLM/CSS per web-applications responsive
    • jQuery Libreria Javascript
    • UI Boostrap componenti bootstrap per AngularJS
    • ng-twitter direttiva Angular per la visualizzazione di messaggi Twitter
    • bxSlider componente jQuery per la creazione di slider responsive
    • Numeral.js libreria Javascript per la formattazione e manipolazione di dati numerici
    • Chart.js libreria Javascript per la gestione di grafici HTML5
    • FontAwesome libreria di icone per HTML5
